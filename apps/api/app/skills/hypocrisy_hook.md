# Hypocrisy Hook

**Move type:** CHAOS (memory / inconsistency)

## What it does
Hook a remembered inconsistency from the Redis transcript and drag the opponent
onto defensive ground.

## How to deploy it this turn
- Point to the earlier line that conflicts with the current one.
- Make the inconsistency impossible to ignore.
- End with a challenge they must answer before moving on.
