# Reframe Attack

**Move type:** CHAOS (disruption / reframing)

## What it does
Refuse the opponent's framing entirely and replace their premise with yours. By
flipping the question that's being asked, you win the argument before it's
scored — the audience now judges on *your* terms, and the opponent's prepared
points suddenly answer a question nobody is asking.

## How to deploy it this turn
- Name the hidden assumption in the opponent's framing.
- Reject it out loud: "That's the wrong question."
- Substitute the question you can win, then answer it decisively.

## Example
"You're asking whether we can afford to act. Wrong question. The real question
is whether we can afford the bill that arrives if we don't — and that bill is
ten times larger and already overdue."
