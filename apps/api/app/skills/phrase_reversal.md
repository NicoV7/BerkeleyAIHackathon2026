# Phrase Reversal

**Move type:** RHETORIC (memory / inversion)

## What it does
Invert the opponent's own remembered phrase so it now supports your side.

## How to deploy it this turn
- Pick one memorable phrase from the transcript.
- Reverse its meaning without losing the audience.
- End with a polished line the agent would actually say.
