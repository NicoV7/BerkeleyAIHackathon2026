# Premise Recall

**Move type:** SOCRATIC (memory / premise)

## What it does
Recall the premise the opponent already relied on and force them to apply it
consistently.

## How to deploy it this turn
- Name the remembered premise.
- Ask whether they still stand by it.
- Show that either answer helps your side.
