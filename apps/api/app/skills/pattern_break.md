# Pattern Break

**Move type:** CHAOS (memory / disruption)

## What it does
Spot a repeated pattern in the Redis transcript, name it, and break the frame so
the opponent's prepared line stops controlling the exchange.

## How to deploy it this turn
- Call out the pattern you remember them using.
- Reject the frame as a trick, dodge, or false choice.
- Replace it with the question your side can win.
