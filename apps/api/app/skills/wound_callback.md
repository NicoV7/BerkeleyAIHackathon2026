# Wound Callback

**Move type:** PATHOS (memory / stakes)

## What it does
Use Redis battle memory to recall a line that minimized harm, then answer by
making the human consequence vivid and immediate.

## How to deploy it this turn
- Refer back to the opponent's remembered wording.
- Put a concrete person inside the consequence.
- Make the emotional stake prove your side, not merely decorate it.
