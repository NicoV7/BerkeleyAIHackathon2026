# Callback Crescendo

**Move type:** RHETORIC (memory / finish)

## What it does
Take a remembered phrase from the Redis transcript and build it into a rhythmic
closing attack in the agent's persona voice.

## How to deploy it this turn
- Reuse the opponent's wording as a callback.
- Escalate it with contrast, repetition, or a rule of three.
- Land on a short final sentence that is easy to remember.
