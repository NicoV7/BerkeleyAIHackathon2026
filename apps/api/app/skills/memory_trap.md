# Memory Trap

**Move type:** SOCRATIC (memory / question)

## What it does
Build a question from a remembered opponent claim. The goal is to make their own
words become the trap they must answer.

## How to deploy it this turn
- Quote or paraphrase the remembered claim.
- Ask one precise question that exposes its hidden assumption.
- Let the question carry the attack; do not over-explain.
