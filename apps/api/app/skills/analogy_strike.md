# Analogy Strike

**Move type:** RHETORIC (style / framing)

## What it does
Map the contested issue onto a vivid, familiar situation where the right answer
is obvious, then carry that intuition back. A good analogy lets the audience
*feel* they already agree with you before the opponent can object.

## How to deploy it this turn
- Choose a simple scenario whose verdict is uncontroversial.
- Draw the structural parallel explicitly, point for point.
- Close the loop: "So just as you'd never X, we shouldn't Y."

## Example
"Refusing to fund prevention because treatment is cheaper today is like skipping
oil changes to save money — right up until the engine seizes on the highway.
We're arguing over the oil change while you're budgeting for a new engine."
