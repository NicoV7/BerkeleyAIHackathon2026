# Contradiction Ledger

**Move type:** LOGOS (memory / evidence)

## What it does
Read the Redis encounter transcript and agent memories as a ledger of claims.
Find one contradiction, missing warrant, or changed standard in the opponent's
recent lines, then turn it into a clean evidentiary attack.

## How to deploy it this turn
- Quote or precisely paraphrase the remembered claim.
- Name the contradiction or unsupported inference.
- Close with the conclusion the opponent must now concede.
